# tpDai – Recipe Search Mobile App

## Description
tpDai is a mobile application developed using React Native (Expo) that allows users to search for food dishes and view detailed recipe information using real data from an external API.

## Features
- Search recipes by name using a search bar
- Fetch real-time data from the Spoonacular API
- Display search results in interactive cards
- View detailed information for selected dishes
- Global state management using React Context API
- Modular and scalable project structure

## Technologies
- React Native (Expo)
- JavaScript (ES6)
- Context API
- Custom React Hooks
- External REST API (Spoonacular)

## Project Structure
- `componentes/` – Reusable UI components
- `pantallas/` – Application screens
- `hooks/` – Custom hooks for business logic
- `contextos/` – Global state management
- `servicios/` – API communication layer

## Learning Focus
This project was built to practice component-based architecture, state management, API integration, and separation of concerns in a mobile application environment.
